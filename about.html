Film Finder

About This Application

Film Finder is a dynamic web application designed to help users easily discover movie information by integrating with the OMDb API. Users can input a movie title or choose from suggested titles to view detailed data like posters, ratings, genres, and plot summaries. The project showcases dynamic features, API interaction, and modern web design practices using HTML, CSS, JavaScript, jQuery, and Bootstrap.
